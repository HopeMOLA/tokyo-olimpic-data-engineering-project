Tokyo Olympic Azure Data Engineering Project
📌 Project Overview

This project demonstrates an end-to-end Azure Data Engineering pipeline using the Tokyo Olympic dataset.

The solution covers:

Data ingestion

Data transformation

Data storage

Data analytics

Cloud-based pipeline orchestration

The objective of this project is to simulate a real-world data engineering workflow using Microsoft Azure services.

🛠️ Tech Stack

Microsoft Azure

Azure Data Factory

Azure Data Lake Storage

Azure Synapse Analytics

Python

Pandas

Jupyter Notebook

Git & GitHub

🚀 How to Run the Project Locally

Clone the repository:

git clone https://github.com/YOUR_USERNAME/tokyo-olympic-azure-data-engineering-project.git


Navigate into the folder:

cd tokyo-olympic-azure-data-engineering-project


Install dependencies:

pip install pandas


Open the Jupyter Notebook:

jupyter notebook
